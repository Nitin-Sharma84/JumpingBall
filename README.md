# Jumping Ball Game 🎮

A 2D endless runner game inspired by Flappy Bird, built using Unity.

## Features
- Smooth player jumping mechanics
- Infinite obstacle spawning
- Score tracking system
- Game Over & Restart system
- Sound effects integration

## Controls
- Spacebar → Jump

## Technologies Used
- Unity (2D)
- C#
- Visual Studio
